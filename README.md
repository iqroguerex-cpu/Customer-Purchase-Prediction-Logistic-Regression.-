## 📌 Overview

This is a simple and interactive **Machine Learning web app** that predicts whether a customer will purchase a product based on:

* 👤 Age
* 💰 Estimated Salary

The app uses a **Logistic Regression model** and provides real-time predictions through a clean Streamlit interface.

---

## 🎯 Features

✔️ Simple and user-friendly UI
✔️ Real-time predictions
✔️ Interactive sliders for input
✔️ Built using Scikit-learn
✔️ Deployed using Streamlit

---

## 🧠 How It Works

1. Dataset is loaded
2. Data is split into training and testing sets
3. Features are scaled using StandardScaler
4. Logistic Regression model is trained
5. User inputs are taken
6. Prediction is displayed instantly
